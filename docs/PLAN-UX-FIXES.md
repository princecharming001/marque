### Craft seed corpus for Fix E (if no web tools)
Retention: >50% avg watch is good for <30s; most drop-off 0-3s; hook in first second ≈ +41% retention; motion in first frame ≈ +35% watch. Structure: condensed AIDA (attention 0-2s / interest 2-8s / desire 8-20s / action last 3-5s). Pacing: no static shot >3s; entertainment 1-2s cuts, education 2-4s; beat-timed cuts ≈ +23% completion. Cut grammar: J-cut (audio leads, momentum) / L-cut (audio lingers, weight); cut on action to hide edits; punch-in on load-bearing lines; pattern interrupts at hook + section boundaries only. Captions: word-by-word or 3-word phrases, high contrast + outline, lower-middle third, 55-75pt @1080×1920. Hook = five simultaneous layers (visual + text overlay + caption + voiceover + audio). Loop-friendly endings flow back into the opening beat.
